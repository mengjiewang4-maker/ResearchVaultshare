# 量子力学演示库

这是一个按 Karpathy-style LLM wiki 方法组织的 Obsidian 知识库。

## 结构 / Structure

- `raw/`: 原始资料层，保存论文、书籍、视频、网页、数据集和媒体资源。
- `wiki/`: 知识整理层，保存概念、数学推导、演示方案、实验记录和综合总结。
- `templates/`: 新建笔记时可复用的模板。
- `AGENTS.md`: 给 AI agent 使用的仓库操作规则。
