# 原始资料 / Raw Materials

这里存放原始来源资料。优先直接添加未经改写的文件，然后在 `wiki/` 中做摘要或综合整理。

## 文件夹 / Folders

- `articles/`: 网页、博客文章、文档导出。
- `papers/`: 学术论文和预印本。
- `books/`: 教材、书籍笔记，以及在允许范围内保存的章节摘录。
- `videos/`: 视频转录稿、链接、截图和字幕。
- `notes/`: 尚未整理提炼的粗略捕捉笔记。
- `datasets/`: 演示或实验使用的数值数据。
- `assets/`: 图片、图表、动画、音频和其他媒体资源。
- `clippings/`: 仍需分类整理的快速剪藏。
